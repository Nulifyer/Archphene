# Support

Archphene is an active prototype. Community support is provided through GitHub Issues rather than private troubleshooting.

Before opening an issue:

1. Read the [README](README.md) and [current limitations](docs/roadmap.md).
2. Search existing issues.
3. Reproduce on the latest release or `main`.
4. Collect the Android version, device model, ABI, Arch package version, and relevant logs.

Use the bug form for reproducible defects and the feature form for scoped proposals. Security reports must follow [SECURITY.md](SECURITY.md).

General Android, Arch Linux, Arch Linux ARM, KDE, GTK, or GrapheneOS support questions that do not involve Archphene should go to those projects' own support channels.
